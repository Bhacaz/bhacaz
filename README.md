## Lib and tools

* **[Docs as code Confluence](https://github.com/Bhacaz/docs-as-code-confluence)** - Publish the content of a folder to Confluence Github Action
* **[Solid APM](https://github.com/Bhacaz/solid_apm)** - A DB base engine for Rails Application Performance Monitoring.
* **[vuerb](https://github.com/Bhacaz/vuerb)** - Frontend framework experimentation to use Ruby standard lib like ERB to build reactive webapp with WASM.
* **[gostacking](https://github.com/Bhacaz/gostacking)** - Git stacking CLI using merge.
* **[BrightSerializer](https://github.com/petalmd/bright_serializer)** - Fast and light Ruby serializer with Grape OpenAPI Entity
* **[GravityMailbox](https://github.com/petalmd/gravity_mailbox)** -  Rails Engine for ActionMailer that aim to make it simple to visualize mail sent by your Rails app directly through your Rails app.
